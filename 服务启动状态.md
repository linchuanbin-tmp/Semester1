## 🔧 如果前端无法启动

可能的原因：
1. Node.js 未安装或未添加到 PATH
2. 前端依赖未安装

**解决方法：**
1. 确保已安装 Node.js (https://nodejs.org/)
2. 进入前端目录安装依赖：
   ```powershell
   cd F:\edgedownload\DjangoProject\DjangoProject\frontend
   npm install
   npm run dev
   ```

## 📝 服务管理

### 停止服务
- 在各个服务窗口按 `Ctrl+C` 停止
- 或使用任务管理器结束进程

### 重新启动服务
- 使用项目根目录下的 `.bat` 启动脚本
- 或按照运行指南手动启动
